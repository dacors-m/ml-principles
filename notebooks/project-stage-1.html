Creacin de Features Polinomiales para Variables Numricas (Grado 2)

Se aplica una estrategia hbrida que combina transformacin polinomial solo en variables numricas continuas, manteniendo las variables categricas sin modificar.

Proceso:

  1. Separacin de variables: Se dividen las variables numricas (temp, atemp, hum, windspeed) de las categricas codificadas

  2. Transformacin polinomial: Se aplica PolynomialFeatures de grado 2 nicamente a las variables numricas, generando trminos cuadrticos e interacciones

  3. Normalizacin: Se estandarizan las features polinomiales con RobustScaler para mantener estabilidad numrica

  4. Recombinacin: Se concatenan las variables numricas transformadas con las variables categricas originales, creando el dataset completo de entrenamiento

Esta aproximacin permite capturar relaciones no lineales en variables climticas sin afectar la interpretacin de las variables categricas.

In[8]:
x_train_num = x_train[['temp', 'atemp', 'hum', 'windspeed']]
x_train_cat = x_train.drop(['temp', 'atemp', 'hum', 'windspeed'], axis="columns")

pol_2 = PolynomialFeatures(degree=2)
x_train_pol2_num = pol_2.fit_transform(x_train_num)

pol_2_scaler = RobustScaler()
x_train_pol2_num = pol_2_scaler.fit_transform(x_train_pol2_num)

x_train_pol2_num = pd.DataFrame(x_train_pol2_num, columns=pol_2.get_feature_names_out(x_train_num.columns))

x_train_pol2 = pd.concat([x_train_pol2_num, x_train_cat.reset_index(drop=True)], axis=1)

Evaluacin del Modelo Polinomial (Grado 2)

Se realizan predicciones sobre el conjunto de prueba y se evala el rendimiento del modelo mediante tres mtricas:

  • RMSE (Root Mean Squared Error): Mide el error promedio de las predicciones, penalizando ms los errores grandes
  • MAE (Mean Absolute Error): Representa el error absoluto promedio en las mismas unidades que la variable objetivo
  • R (Coeficiente de determinacin): Indica qu porcentaje de la variabilidad de la demanda es explicado por el modelo (valores cercanos a 1 indican mejor ajuste)
